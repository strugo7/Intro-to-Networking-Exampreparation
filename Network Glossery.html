ARP

Protocol Layer 2/3

Address Resolution Protocol

flag English Definition

A communication protocol used for discovering the link layer address, such as a MAC address, associated with a given internet layer address, typically an IPv4 address. This mapping is a critical function in the Internet Protocol Suite.

Hebrew Definition translate

פרוטוקול המיפוי (ARP) הוא פרוטוקול תקשורת המשמש לגילוי כתובת שכבת הקישור, כגון כתובת MAC, המשוייכת לכתובת שכבת האינטרנט נתונה, בדרך כלל כתובת IPv4. מיפוי זה הוא פונקציה קריטית בחבילת פרוטוקולי האינטרנט.
