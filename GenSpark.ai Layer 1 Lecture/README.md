# שיעור אינטראקטיבי — השכבה הפיזית (OSI Layer 1)

## 🎯 מטרת הפרויקט
שיעור אינטראקטיבי ויזואלי שמסביר על השכבה הראשונה (Physical Layer) במודל OSI. 
השיעור כולל הסברים מפורטים, אנימציות קנבס, דיאגרמות אינטראקטיביות ובוחן ידע.

## ✅ תכונות שהושלמו

### 📖 תוכן לימודי
- **מבוא** — הסבר על מודל OSI ומגדל 7 השכבות (אינטראקטיבי)
- **סיבים אופטיים** — מבנה, עקרון פעולה, Single Mode vs Multi Mode, יתרונות וחסרונות
- **כבלי נחושת** — UTP, STP, Coaxial, קטגוריות Ethernet, אותות חשמליים
- **WiFi** — גלי רדיו, תדרי WiFi, דורות WiFi, סימולציית עוצמת אות
- **השוואה** — טבלת השוואה מקיפה, תרחישי שימוש (כרטיסיות מתהפכות), מרוץ מהירויות
- **בוחן ידע** — 10 שאלות עם משוב מיידי, ניקוד וסיכום ביצועים

### 🎨 אנימציות אינטראקטיביות
- אנימציית חלקיקי רשת בדף הבית (Hero Canvas)
- הדגמת שליחת ביטים עם אות דיגיטלי
- סימולציית סיב אופטי — Single Mode ו-Multi Mode
- חתך רוחב אינטראקטיבי של סיב אופטי (hover tooltip)
- אות חשמלי בכבל נחושת (עם סליידרים למתח ותדירות)
- שרטוטי חתכי רוחב של UTP, STP ו-Coaxial
- סימולציית WiFi — שידור גלי רדיו ממכשירים
- סימולציית עוצמת אות WiFi (drag & drop)
- גלי תדר אנימטיביים (2.4GHz, 5GHz, 6GHz)
- מרוץ מהירויות בין הטכנולוגיות

### 🧭 ניווט
- תפריט צד קבוע עם אינדיקטור התקדמות
- ניווט מקלדת (חצים)
- ניווט Swipe (מגע)
- פס התקדמות עליון
- כפתורי הבא/הקודם בכל עמוד

### 📱 רספונסיבי
- עיצוב מותאם לנייד, טאבלט ומחשב
- תפריט צד מתקפל במסכים קטנים
